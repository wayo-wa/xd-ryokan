## XDデザインカンプからのコーディング  
PC版デザインカンプからコーディングしました。
  
担当：コーディング  
使用技術：HTML, Sass(CSS), JavaScript (jQuery/gulp)  
デザインツール：XD  
  
・スクロール時のheader変化（固定、背景色）  
・ファーストビューのアニメーション（css animationでズームしながら画像切り替え）  
・要素をふわっと表示（aos.js）  
・宿泊予約モーダル（flatpickr）  
・ダブの切り替え（jQuery）  
・パンくずリスト  
・ハンバーガーメニュー  
・レスポンシブ対応
